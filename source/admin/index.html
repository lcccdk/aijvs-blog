📖 快速指南

发布新文章的方式


方式 1：通过 GitHub（推荐）

  1. 访问 GitHub 仓库
  2. 导航到 source/_posts/ 目录
  3. 创建新文件：文章标题.md
  4. 使用 Front Matter 格式编写内容
  5. 提交后自动触发 Cloudflare 构建

方式 2：本地 Hexo

# 创建文章
hexo new post "文章标题"

# 编辑文章内容
code source/_posts/文章标题.md

# 推送到 GitHub
git add .
git commit -m "Add: 文章标题"
git push

Cloudflare Pages 会在 2-5 分钟内自动构建并部署。
